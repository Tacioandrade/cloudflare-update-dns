# 変更履歴

Cloudflare DNS Manager の重要な変更はすべてこのファイルに記録されます。

## [2.0.0] - 2026-07-14
### 追加 / 変更
- Cloudflare API から受信したドメインを段階的に読み込みます。
- ポルトガル語、英語、フランス語、スペイン語、簡体字中国語、日本語に対応。設定は保存され、既定ではシステム言語を使用します。
- バージョン履歴をローカライズし、言語ごとにキャッシュします。

## [1.1.4] - 2026-06-27
- Cloudflare DNS Manager への改称と、DNS レコードを作成する `Ctrl + N` ショートカットを追加しました。

## [1.1.3] - 2026-06-26
- Windows ランタイムと Linux・Windows パッケージの CI/CD を修正しました。
## [1.1.2] - 2026-06-26
- DNS 検証、TXT とプロキシの修正、Linux・Windows の自動公開を追加しました。
## [1.1.1] - 2026-06-25
- フォーカス、生体認証、キーボードショートカットを改善しました。
## [1.1.0] - 2026-06-25
- Windows と Linux のフローを文書化し、秘密情報を安全なストレージへ移しました。
## [1.0.8] - 2026-06-24
- アプリ再起動後にログアウトを必須にしました。
## [1.0.7] - 2026-06-24
- ライト、ダーク、システムテーマと Flutter コントロールのローカライズを追加しました。
## [1.0.6] - 2026-06-24
- CDN キャッシュ削除と必要な権限のドキュメントを追加しました。
## [1.0.5] - 2026-06-24
- 初回アクセス時のパスワード作成とユーザー名フィールドの削除を行いました。
## [1.0.4] - 2026-06-19
- DNS 種類セレクターとアプリ内変更履歴表示を追加しました。
## [1.0.3] - 2026-06-19
- ネイティブ生体認証とサブドメイン編集を追加しました。
## [1.0.2] - 2026-06-18
- 自動ページネーション、視覚調整、標準化アイコンを追加しました。
## [1.0.1] - 2026-06-18
- Android ビルド、Kotlin、NDK、アプリケーション ID を修正しました。
## [1.0.0] - 初回リリース
- ブラウザーと Android 向け Cloudflare DNS 管理と動的テーマを提供しました。
