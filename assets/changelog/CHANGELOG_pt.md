# Changelog

Todas as alterações notáveis do Cloudflare DNS Manager serão documentadas neste arquivo.

## [2.0.0] - 2026-07-14
### Adicionado / Alterado
- Carregamento progressivo de domínios recebidos da API Cloudflare.
- Idiomas: português, inglês, francês, espanhol, chinês simplificado e japonês; a preferência é persistente e o idioma do sistema é usado por padrão.
- Histórico de versões localizado e mantido em cache por idioma.

## [1.1.4] - 2026-06-27
- Projeto renomeado para Cloudflare DNS Manager e adicionado atalho `Ctrl + N` para criar um registro DNS.

## [1.1.3] - 2026-06-26
- Corrigido o runtime do Windows e ajustado o CI/CD para os pacotes Linux e Windows.

## [1.1.2] - 2026-06-26
- Validação de registros DNS, correções para TXT e Proxy, e publicação automática para Linux e Windows.

## [1.1.1] - 2026-06-25
- Melhorias de foco, biometria e atalhos `Ctrl + F`, `Ctrl + N`, `Esc` e `Enter`.

## [1.1.0] - 2026-06-25
- Fluxos documentados de build/teste para Windows e Linux; segredos migrados para armazenamento seguro.

## [1.0.8] - 2026-06-24
- Logout obrigatório ao reiniciar o aplicativo.
## [1.0.7] - 2026-06-24
- Temas claro, escuro e do sistema; localização dos controles nativos do Flutter.
## [1.0.6] - 2026-06-24
- Limpeza de cache CDN e documentação das permissões necessárias.
## [1.0.5] - 2026-06-24
- Primeiro acesso para criação de senha e remoção do campo de usuário.
## [1.0.4] - 2026-06-19
- Seletor de tipos DNS e visualização do changelog no aplicativo.
## [1.0.3] - 2026-06-19
- Biometria nativa e edição de subdomínios.
## [1.0.2] - 2026-06-18
- Paginação automática, identidade visual e ícones padronizados.
## [1.0.1] - 2026-06-18
- Correções de build Android, Kotlin, NDK e identificador do pacote.
## [1.0.0] - Lançamento inicial
- Gerenciamento DNS Cloudflare para navegador e Android, com tema dinâmico.
