# 更新日志

此文件记录 Cloudflare DNS Manager 的所有重要更改。

## [2.0.0] - 2026-07-14
### 新增 / 更改
- 从 Cloudflare API 接收域名时进行渐进式加载。
- 支持葡萄牙语、英语、法语、西班牙语、简体中文和日语；语言偏好会持久保存，默认使用系统语言。
- 版本历史已本地化，并按语言缓存。

## [1.1.4] - 2026-06-27
- 更名为 Cloudflare DNS Manager，并新增 `Ctrl + N` 创建 DNS 记录快捷键。

## [1.1.3] - 2026-06-26
- 修复 Windows 运行时以及 Linux 和 Windows 软件包的 CI/CD。
## [1.1.2] - 2026-06-26
- DNS 验证、TXT 和代理修复，以及 Linux 和 Windows 自动发布。
## [1.1.1] - 2026-06-25
- 改进焦点、生物识别和键盘快捷键。
## [1.1.0] - 2026-06-25
- 记录 Windows 和 Linux 流程；机密信息移至安全存储。
## [1.0.8] - 2026-06-24
- 重启应用后必须重新登录。

## [1.0.7] - 2026-06-24
- 浅色、深色和系统主题；Flutter 控件本地化。

## [1.0.6] - 2026-06-24
- CDN 缓存清除和所需权限文档。

## [1.0.5] - 2026-06-24
- 首次访问时创建密码并移除用户名字段。

## [1.0.4] - 2026-06-19
- DNS 类型选择器和应用内更新日志视图。

## [1.0.3] - 2026-06-19
- 原生生物识别和子域名编辑。

## [1.0.2] - 2026-06-18
- 自动分页、视觉调整和标准化图标。

## [1.0.1] - 2026-06-18
- Android 构建、Kotlin、NDK 和应用标识符修复。

## [1.0.0] - 初始版本
- 面向浏览器和 Android 的 Cloudflare DNS 管理及动态主题。
