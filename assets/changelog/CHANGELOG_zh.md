# 更新日志

GitHub 上的项目：[https://github.com/Tacioandrade/cloudflare-update-dns/](https://github.com/Tacioandrade/cloudflare-update-dns/)

本项目的所有重要变更都将记录在此文件中。

## [2.0.1] - 2026-07-19
### 新增
- 在记录页面中新增可组合的 DNS 记录类型和代理状态筛选器，并使用“设置”中启用的记录类型。

## [2.0.0] - 2026-07-14
### 新增 / 更改
- 域名列表现在会在从 Cloudflare API 接收到每个域名后逐步显示，无需等待所有分页加载完成。
- 新增对葡萄牙语、英语、法语、西班牙语、简体中文和日语的支持，并可持久保存语言选择及检测系统语言。
- 版本历史现在会以应用程序中选择的语言显示。

## [1.1.4] - 2026-06-27
### 新增 / 更改
- 项目名称由 Cloudflare Update DNS 更改为 Cloudflare DNS Manager。
- 新增键盘快捷键 `Ctrl + N`，用于打开新建 DNS 记录界面。

## [1.1.3] - 2026-06-26
### 修复
- 修复了 Windows 版本因缺少 Microsoft Visual C++ Runtime 而产生的运行错误。
- 修正了项目的 CI/CD 设置，使其仅发布当前版本的更新日志，并将 Windows 和 Linux bundle 打包到 `cloudflare-update-dns` 文件夹中。

## [1.1.2] - 2026-06-26
### 新增 / 修复
- 新增对创建和编辑 DNS 记录时所输入内容的验证。
- 对 `A` 记录验证 IPv4，对 `AAAA` 记录验证 IPv6。
- 对 `CNAME`、`MX` 和 `NS` 记录验证域名格式，且不依赖固定的 TLD 列表。
- 对 `SRV` 记录验证基本格式，并验证 `TXT` 记录的内容不为空。
- 修复域名验证，拒绝在 `CNAME`、`MX`、`NS` 记录及 `SRV` 目标中输入 IP 地址。
- 修复 DMARC 和 DKIM 等 `TXT` 记录的创建逻辑，不再向不支持代理的记录类型错误发送 `proxied` 字段。
- `Proxied` 控件现在仅对 `A`、`AAAA` 和 `CNAME` 记录显示。
- 调整了 Cloudflare API 拒绝创建或更新记录时显示的错误消息。
- 启用项目 CI/CD，自动生成 Windows 和 Linux 应用程序。

## [1.1.1] - 2026-06-25
### 新增 / 修复
- 打开应用程序时，键盘焦点会根据所配置的身份验证类型定位到密码或生物识别字段。
- 新增键盘快捷键 `Ctrl + F`，用于快速打开域名列表和 DNS 记录列表中的搜索。
- 新增键盘快捷键 `ESC`，用于快速关闭搜索功能。
- 新增键盘快捷键 `ESC`，用于快速关闭 DNS 记录创建/编辑界面。
- 新增键盘快捷键 `ESC`，用于从 DNS 记录创建/编辑界面或设置界面返回域名列表。
- 在登录界面的密码字段中按下 `Enter` 键时自动登录。
- 创建/编辑 DNS 记录时，在子域名和内容字段中按下 `Enter` 键即可自动保存记录。

## [1.1.0] - 2026-06-25
### 新增
- 新增 Windows 版本的文档及构建/测试流程。
- 新增 Linux 版本的文档及构建/测试流程。
### 安全
- 将 Cloudflare API Token 的存储方式更改为 `flutter_secure_storage`，使用各平台的原生安全存储。
- 将应用密码改为使用随机盐的 PBKDF2-HMAC-SHA256 哈希并存入安全存储，不再以明文持久保存密码。
- 已通过身份验证的会话仅保存在内存中，重新打开应用程序时需要再次登录。
- `shared_preferences` 中仅保留主题和 DNS 记录类型等非敏感设置。

## [1.0.8] - 2026-06-24
### 修复
- 关闭并重新打开应用程序时强制退出登录，在所有操作系统上都需要重新登录。

## [1.0.7] - 2026-06-24
### 新增 / 修复
- 新增浅色和深色主题支持，并自动检测操作系统中配置的主题。
- 在设置中新增系统、浅色或深色主题选择项。
- 新增 Flutter 原生本地化，以设备语言显示复制、粘贴和全选等内部文本。

## [1.0.6] - 2026-06-24
### 新增 / 修复
- 在 DNS 记录界面新增清除所选区域 CDN 缓存的功能。
- 记录了使用自定义 Cloudflare Token 清除 CDN 缓存所需的权限。

## [1.0.5] - 2026-06-24
### 新增 / 修复
- 首次访问时直接打开密码创建界面，无需默认密码或生物识别验证。
- 从登录界面移除用户名字段；现在仅可通过已注册密码或生物识别验证访问。

## [1.0.4] - 2026-06-19
### 新增 / 修复
- 在设置中新增 DNS 记录类型选择器，用于筛选系统加载的记录。
- 新增在应用程序内直接查看更新日志的界面。

## [1.0.3] - 2026-06-19
### 新增 / 修复
- 实现原生生物识别登录（指纹 / 人脸识别），并提供透明且自动的回退机制。

## [1.0.2] - 2026-06-18

- 对界面和视觉识别进行了细微调整。
- 实现新徽标并生成标准化图标（Android、iOS 和 Web）。

## [1.0.1] - 2026-06-18
### 修复
- 解决兼容性问题并修复 Android 原生构建代码。
- 调整 Kotlin 版本和 NDK 配置，以确保可靠编译。
- 将软件包标识符（Application ID）从示例默认值更改为正式的企业签名。
