# Changelog

All notable Cloudflare DNS Manager changes are documented in this file.

## [2.0.0] - 2026-07-14
### Added / Changed
- Progressive loading for domains received from the Cloudflare API.
- Portuguese, English, French, Spanish, Simplified Chinese, and Japanese; the preference is persistent and the system language is used by default.
- Version history is localized and cached per language.

## [1.1.4] - 2026-06-27
- Renamed to Cloudflare DNS Manager and added `Ctrl + N` to create a DNS record.

## [1.1.3] - 2026-06-26
- Fixed the Windows runtime and adjusted CI/CD for Linux and Windows packages.

## [1.1.2] - 2026-06-26
- DNS record validation, TXT and proxy fixes, and automatic Linux and Windows publishing.

## [1.1.1] - 2026-06-25
- Focus, biometric, and `Ctrl + F`, `Ctrl + N`, `Esc`, and `Enter` shortcut improvements.

## [1.1.0] - 2026-06-25
- Documented Windows and Linux build/test flows; secrets moved to secure storage.

## [1.0.8] - 2026-06-24
- Logout is required after restarting the application.

## [1.0.7] - 2026-06-24
- Light, dark, and system themes; Flutter native control localization.

## [1.0.6] - 2026-06-24
- CDN cache purge and required-permission documentation.

## [1.0.5] - 2026-06-24
- First-use password setup and removal of the username field.

## [1.0.4] - 2026-06-19
- DNS type selector and in-app changelog view.

## [1.0.3] - 2026-06-19
- Native biometric authentication and subdomain editing.

## [1.0.2] - 2026-06-18
- Automatic pagination, visual refinements, and standardized icons.

## [1.0.1] - 2026-06-18
- Android build, Kotlin, NDK, and application-ID corrections.

## [1.0.0] - Initial release
- Cloudflare DNS management for browser and Android, with a dynamic theme.
